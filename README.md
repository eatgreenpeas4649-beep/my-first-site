# my-first-site
AIツールナビは、AI初心者から上級者向けの総合情報ポータルサイトです。「何を作りたいか？」を選択するだけでChatGPT、Claude、Gemini、Perplexityなど最適AIを診断提案。比較表で得意分野・料金・日本語対応を一目で確認でき、無料/画像生成などのフィルタ機能も完備。  最新AIニュース、商用利用ガイド、コピペ可能な高品質プロンプト集、生成速度検証データを掲載。ダークモード対応で長時間作業も快適。コミュニティ機能として掲示板、用語集、AIスキル検定クイズ、ブックマークも実装。  同じプロンプトでの全AI比較検証や、API連携・ローカルLLM構築ガイドなど実践コンテンツも充実。
